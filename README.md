# The Hobbit Cutscene Editor

Incredibly useful tool to modify in-game cutscenes with ease! Create shots, syncpoints and actions in the nice Web UI.

## Usage

Get the latest versin from [Releases](https://github.com/hobbit-kingdom/hobbit-cutscene-editor/releases) tab, unpack the archive and simply open the *index.html*. The app should open inside your web browser.

# Main Features
  - Allows you to easily create new **Shots**, **Syncpoints** and all types of **Actions** to make the process of cutscene creation as easy as it can be.
    
  - Shows **tooltips** with explanation of the fields and displays only the most important fields that you might need to modify.
    
  - Built in **text-preview** of the generated cutscene (as you see it the .EXPORT files).
    
  - Ability to work on **multiple cutscenes** at the same time.
    
  - **Load** a Cinemas/Cutscenes .export from the game to get access to all of the cutscenes.
    
  - **Copy** or **download** your brand new cutscenes.
    
  - Camera creation tab with **Keyframes** and **Path** generation (in BETA!).
    
  - **Complicated options**: Random Mod (bunch of effects from king174rus youtube challanges), PickUpAll Mod by AiOne, Kill All Mod.
    
  - **Skinchanger**: ability to change Bilbos Skin Texture and Props models/textures (Stick, Stone, Sword)
